逐项审查方案，直到关键问题都有明确结论。沿着每个决策分支检查约束、依赖和取舍；每提出一个问题，都同时给出推荐答案及理由。

每次只问一个问题，在得到针对该问题的反馈之前不要继续。一次抛出多个问题会让人无所适从。

如果某个问题可以通过探查代码库来回答，那就去探查代码库，而不是发问。
